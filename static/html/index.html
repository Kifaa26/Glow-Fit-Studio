Home Page
HTTP Methods Path Description
GET / Fetches landing/home page

Users
HTTP Method Path Description
GET /users Fetches all Users data from the database
GET /user/:id Fetches a single User's data from the database for the id provided
POST /user/register Adds a new user to the database.
PATCH /user/update/:id Updates a user for the id value provided.
DELETE /user/delete/:id Deletes a user's data from the database for the id provided

Instructors
HTTP Method Path Description
GET /instructors Fetches all instructors data from the database
GET /instructor/:id Fetches a instructor's data from the database for the id provided.
GET /instructor/latest Fetches the last 4 instructors added to the database from the most recent.
POST /instructor/addInstructors Adds a new instructor to the database.
PATCH /instructor/update/:id Updates a instructor for the id value provided.
DELETE /instructor/delete/:id Deletes a instructor's data from the database for the id provided
